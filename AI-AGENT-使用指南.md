# 价值投资研究：AI Agent 使用指南

用 Cursor / 其他 AI Agent 辅助整理年报、资讯并深化公司研究。按「资料整理 → 理解 → 输出」分阶段使用。

---

## 一、资料整理阶段

### 1. 年报

- **你可以做的**：把年报 PDF 放到 `年报/[公司代码]/`，或保存巨潮/上交所/港交所等年报链接到该目录下的 `链接.txt` 或 Markdown。
- **让 AI 帮忙**：
  - 「请根据这份年报 [粘贴章节或上传 PDF 选段]，提炼：主营业务、主要产品/服务、前五大客户与供应商占比、毛利率/净利率趋势、现金流与资本开支、管理层对风险的表述。」
  - 「把第 X 节的财务数据整理成表格（营收、利润、ROE、负债率、经营现金流）。」
  - 「用中文写一段 200 字左右的年报摘要，适合放进研究笔记。」

### 2. 资讯（新闻、研报、公告）

- **你可以做的**：在 `资讯/按公司/[公司]/` 或 `资讯/按主题/` 下用 Markdown 记录标题、来源、链接、日期，必要时粘贴关键段落。
- **让 AI 帮忙**：
  - 「根据这几条资讯 [粘贴]，归纳：事件、对业务/财务的可能影响、需要后续跟踪的点。」
  - 「把同一公司最近一个月的新闻按「利好/利空/中性」简单分类，并标出需要和年报交叉验证的表述。」
  - 「从这段研报摘录中提取：目标价、核心假设、主要风险提示。」

---

## 二、理解与交叉验证

- **让 AI 帮忙**：
  - 「我年报里看到 [某数据或表述]，资讯里提到 [某说法]，两者是否一致？有哪些地方需要再查证？」
  - 「用「波特五力」或「护城河」框架，根据我提供的年报摘要和资讯，列出要点和待验证问题。」
  - 「根据年报和资讯，列出这家公司 3 个主要风险和 3 个需要持续跟踪的指标。」

---

## 三、输出：研究笔记与清单

- **你可以做的**：在 `研究笔记/[公司代码]/` 下用 Markdown 写你的理解；在 `模板与清单/` 里放通用清单（如「公司研究检查清单」）。
- **让 AI 帮忙**：
  - 「根据我这份零散笔记 [粘贴]，整理成：公司概况、业务与竞争、财务要点、风险与跟踪、当前结论与疑问。」
  - 「按我提供的清单 [粘贴清单]，检查我的研究笔记是否覆盖了每一项，并标出缺失或薄弱部分。」
  - 「根据我给的营收、利润、现金流数据，写一段简单的 DCF/市盈率估值逻辑说明（注明假设和局限性）。」

---

## 四、和 Cursor 协作的实用技巧

1. **@ 引用文件**：在对话里用 `@年报/某某公司/摘要.md`、`@研究笔记/某某公司/笔记.md`，让 Agent 基于你已有文档回答，避免空泛结论。
2. **小步迭代**：先让 AI 整理「一家公司一年报」的摘要，你校对后再扩展为多年度、多公司。
3. **保留来源**：让 AI 在摘要/表格里注明「据年报第 X 页」或「据 XX 报道」，方便日后核对。
4. **清单驱动**：在 `模板与清单/` 里维护一份「公司研究检查清单」，每次研究新公司时让 AI 按清单帮你查漏补缺。

---

## 五、注意事项

- AI 的输出需要你**自行核对原始资料**，尤其是数字和法规相关表述。
- 估值、买卖决策务必由你自己做，AI 只辅助整理与框架，不替代独立判断。
- 年报、公告等以交易所/公司官网为准；资讯建议保留链接和日期便于溯源。

---

按上述方式，你可以把「年报 + 资讯 + 研究笔记」串成一条线，用 AI Agent 做整理、归纳和结构化，把时间更多花在理解和决策上。
