# 公司研究检查清单

研究一家公司时，可对照本清单，确保不遗漏关键维度。完成一项打勾或简要备注。

## 一、业务理解

- [ ] 主营业务与主要产品/服务是什么？
- [ ] 收入按产品/地区/客户的构成？（有无集中度风险）
- [ ] 上下游与主要客户、供应商情况？
- [ ] 所处行业格局、主要竞争对手、公司相对位置？
- [ ] 管理层在年报/交流中对战略和风险的表述？

## 二、财务要点

- [ ] 最近 3–5 年营收、利润、毛利率、净利率趋势？
- [ ] ROE、资产负债率、有息负债与现金流匹配？
- [ ] 经营现金流与净利润的关系？资本开支与自由现金流？
- [ ] 有无大额商誉、减值、非经常性损益需单独考虑？

## 三、资讯与跟踪

- [ ] 近期重要公告、新闻是否已阅读并做摘要？
- [ ] 与年报表述是否一致？有无需要后续验证的说法？
- [ ] 列出 2–3 个需要持续跟踪的指标或事件

## 四、风险与结论

- [ ] 主要风险（业务、财务、治理、政策等）列出并简要评估
- [ ] 当前结论：关注/观望/排除，及理由
- [ ] 未解疑问与下一步要查的资料

---

*可将本清单复制到具体公司的研究笔记中，按项填写；也可让 AI 根据你的笔记对照清单查漏补缺。*
