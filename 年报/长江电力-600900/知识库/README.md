# 长江电力年报知识库

本目录用于把 `2003-2024` 全部年报沉淀为后续可直接引用的本地知识库，避免重复重解析 PDF。

## 目录说明

- `年报全文/`：每份年报的完整 Markdown 全文，按 PDF 页码分段。
- `结构化/`：每年一份 JSON，包含 PDF 哈希、页统计、目录索引、顶层节标题、Excel 对比摘要。
- `索引.md`：人工可读的总索引。
- `索引.json`：程序可读的总索引。
- `校验报告.md`：完整校验结果。

## 总体结论

- 年报数量：`22`
- PDF 总页数：`3616`
- Excel 对比累计：一致 `2409` 条，差异 `0` 条，未命中 `118` 条
- 后续引用建议：优先读 `索引.md` 定位年度，再打开对应 `年报全文/*.md` 或 `结构化/*.json`。
