# 长江电力投资收益专题（2003-2024）

最后更新：2026-03-21

## 一、这份专题回答什么问题

这份笔记只回答一件事：

**长江电力上市以来，投资收益到底是怎么来的，为什么会从利润表里的“小配角”，慢慢变成越来越像“第二利润引擎”的存在。**

核心不是只看某一年投资收益高不高，而是看三件事：

- 这笔收益主要来自什么；
- 它是一次性的，还是可持续的；
- 它对主业利润到底是“增厚”，还是已经接近“平台化的第二来源”。

## 二、口径说明

- 数据优先采用 `2003-2024` 各年年报中 `合并利润表附注` 的 `投资收益` 明细。
- 金额统一换算为 `亿元`，四舍五入保留两位小数。
- 老年报和新年报科目口径不同：
  - `2003-2006` 主要看到 `股权投资收益`、`债权投资收益`、`基金投资收益`；
  - `2007-2018` 开始更多看到 `权益法收益`、`可供出售金融资产持有/处置收益`；
  - `2019` 以后逐步切换为 `权益法收益`、`其他权益工具股利`、`其他非流动金融资产收益` 等新准则口径。
- `2016`、`2023` 涉及同一控制下企业合并追溯重列，因此看趋势时更适合看“构成变化”，不宜机械做单年同比。

## 三、先给一句话结论

长江电力投资收益的历史，大致经历了四个阶段：

1. `2003-2005`：零散财务投资阶段，只是利润表配角。
2. `2006-2011`：金融资产处置波动阶段，一次性特征明显。
3. `2012-2020`：长期股权收益抬升阶段，开始出现“第二利润引擎”雏形。
4. `2021-2024`：联营企业收益经常化阶段，投资收益越来越平台化，但仍不能等同于主业利润。

## 五、按阶段看，投资收益是怎么变的

### 1. `2003-2005`：只是利润表的小配角

这一阶段的投资收益只有 `0.12-0.77` 亿元量级。

特征很简单：

- 规模小；
- 主要来自股权、基金、债券等财务性投资；
- 对净利润不是决定性变量。

这时的长江电力，利润表核心仍然是水电主业，投资收益只是辅助项。

### 2. `2006-2011`：金融资产处置驱动，波动很大

这一阶段投资收益开始“看起来很重要”，但要非常注意质量差异。

最典型的是：

- `2006`：出售建行股份，投资收益突然跳到 `8.66` 亿元；
- `2007`：再次出售建行股份，投资收益冲到 `20.95` 亿元；
- `2008`：金融市场转弱，投资收益又回落到 `4.32` 亿元；
- `2009`：出售中国国航、中信银行股票，投资收益回到 `10.75` 亿元；
- `2011`：出售建行 H 股、转让大冶有限股权，投资收益再到 `11.17` 亿元。

这说明这一段的投资收益有两个特征：

- 对利润表影响已经不小；
- 但一次性很强，受金融资产处置时点影响很大。

所以，这一阶段还不能把投资收益理解为“第二主业”，更像是：

`主业利润 + 较大波动的财务性增厚`

### 3. `2012-2020`：权益法收益逐步坐大

从 `2012` 开始，一个更重要的变化出现了：

**投资收益越来越不是靠“卖掉什么”，而是靠“持有什么”。**

表现为：

- `2012-2014`：权益法收益从 `5.52` 亿元提升到 `7.24-8.46` 亿元区间；
- `2015-2018`：权益法收益升到 `10-14` 亿元区间；
- `2019-2020`：权益法收益再上台阶，先后到 `20.77` 亿元、`31.83` 亿元。

这背后反映的不是交易技巧，而是：

- 联营企业和长期股权投资布局在变厚；
- 被投企业盈利能力在增强；
- 长江电力本身越来越像一个大型能源资本平台。

因此，到 `2019-2020`，说投资收益是“第二利润引擎”，已经开始成立。

### 4. `2021-2024`：越来越平台化，但仍有单年波动

这一阶段最容易误判。

一方面，投资收益的确很高：

- `2021`：`54.26` 亿元；
- `2022`：`46.00` 亿元；
- `2023`：`47.50` 亿元；
- `2024`：`52.58` 亿元。

另一方面，不能把这几年的高投资收益全都当成“完全经常性”。

原因在于：

- `2021` 明显受处置上海电力、四川川投能源等股票影响，一次性较强；
- `2022` 因上一年的一次性高基数而回落；
- `2023-2024` 才更接近新的稳定结构，即以联营、合营企业收益为大头。

所以，更准确的判断是：

- `2021`：高，但偏一次性；
- `2022`：回落，但底盘未坏；
- `2023-2024`：真正开始体现“经常化平台收益”的味道。

## 六、为什么说它越来越像“第二主业”

这个说法成立，主要是因为投资收益的来源已经发生了结构性变化。

过去更像：

- 卖股票；
- 卖金融资产；
- 做财务性投资。

现在更像：

- 联营企业按权益法确认收益；
- 持有型股权资产持续分红；
- 长期投资组合稳定贡献回报。

这两种投资收益的质量完全不同。

前者偏交易型，波动大，偶发性强。

后者偏平台型，虽然不等同于主业经营利润，但明显更稳定，也更能反映公司资本配置能力。

## 七、但它为什么还不能等同于真正的“第二主业”

虽然投资收益越来越重要，但我还是不建议把它直接等同于水电主业。

原因有三点：

1. 它的可控性弱于主业经营利润。
   主业利润更多取决于来水、调度、电价、成本和资本结构；投资收益还会受到被投企业经营、资本市场、股权处置时点影响。

2. 它的稳定性虽在增强，但仍有单年波动。
   `2021` 和 `2022` 的对比就很典型，高位年份里仍然包含明显的一次性成分。

3. 它对现金流的转化路径不完全一样。
   权益法收益会先体现在利润表，不一定立刻变成上市公司当期现金流；这一点和电站经营利润不同。

所以，更稳妥的说法是：

**投资收益已经从“偶发性非主业补充”，变成“长期资本平台的第二利润来源”，但还没有强到可以和大水电主业并列为完全同质的两条主业线。**

## 八、现阶段研究判断

如果把这个专题压缩成几句话，我会这样写：

1. 长江电力投资收益的历史，前半段看交易和处置，后半段看长期股权和平台配置。
2. `2006`、`2007`、`2009`、`2011`、`2021` 这些年份，一次性特征更强，分析时要降权。
3. `2019` 以后，尤其 `2023-2024`，投资收益的大头越来越稳定地来自联营、合营企业和持有型股利，质量明显提升。
4. 因此，“投资收益越来越像第二主业”这个判断，大方向是成立的；但更严谨的表达应当是：

**它越来越像“平台化的第二利润引擎”，而不是完全等同于发电主业的第二主业。**
