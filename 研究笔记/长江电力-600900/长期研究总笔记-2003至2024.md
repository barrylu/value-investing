## 二、一句话总判断

长江电力上市以来最重要的事情，不是某一年的利润高低，而是：

**它一步步把长江干流最优质的大型水电资产装进上市公司，把招股书里的“水电资产平台化”故事，兑现成了一个长期高分红、强现金流、强壁垒的公用事业核心资产。**

## 三、压缩版主线

如果把 `2003-2024` 浓缩成一条主线，可以写成：

`招股书平台逻辑 -> 三峡机组持续装入 -> 三峡资产进一步整合 -> 溪洛渡/向家坝注入 -> 乌东德/白鹤滩注入 -> 六库联调成型 -> 高分红现金牛`

它对应的不是单一故事，而是三条长期同时推进的线：

1. `资产线`：优质大水电资产持续装入上市平台。
2. `经营线`：平台体量扩大后，利润和现金流中枢不断抬升。
3. `回报线`：高分红从结果逐步变成制度，再变成公司特征。

## 四、拆分后的结构逻辑

这次拆分，核心是把原文件里的内容按层次分开：

1. `总览层`：历史阶段、护城河、投资主线、总体结论。
2. `数据层`：年度营收/净利润/分红总表，股东回报时间轴。
3. `判断层`：当前四个核心问题与阶段性结论。
4. `专题层`：资产效率判断、利润表演变。

这样以后更新时可以按主题维护：

## 五、当前最值得跟踪的 4 条线

后续如果只盯少数几个指标，优先盯这四组：

1. `发电量 / 来水 / 利用小时`：看六库联调是否持续提升经营韧性。
2. `经营现金流 / 资本开支 / 自由现金流`：看高分红是否有真实现金支撑。
3. `财务费用 / 资产负债率 / 融资成本`：看并表后的财务消化是否顺利。
4. `分红总额 / 分红率 / 中期分红安排`：看高分红是否从阶段性承诺走向长期制度习惯。
