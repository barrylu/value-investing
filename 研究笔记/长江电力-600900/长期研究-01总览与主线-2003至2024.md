# 长江电力长期研究-01：总览与主线（2003-2024）

最后更新：2026-03-21

## 一、一句话结论

长江电力上市以来最重要的事情，不是某一年的利润高低，而是：

**它一步步把长江干流最优质的大型水电资产装进上市公司，把招股书里“水电资产平台化”的故事，兑现成了一个长期高分红、强现金流、强壁垒的公用事业龙头。**

## 二、用 4 个阶段看 `2003-2024`

### 阶段一：`2003-2008`
### 从“葛洲坝 + 三峡首批机组”到上市平台成型

这是长江电力最早的资本市场起点。

#### 1. 招股书阶段的核心逻辑

招股书已经把公司未来写得很清楚：

- 先以上市募集资金收购三峡首批机组；
- 再持续收购后续投产机组；
- 最终把长江干流优质发电资产不断装入上市公司；
- 通过装机增长、发电量增长、电价和调度优化来释放利润。

也就是说，长江电力从上市第一天起，就不是普通电力股，而是**三峡体系优质水电资产的资本市场承载平台**。

#### 2. 早期兑现情况

从 `2003-2005` 年报看：

- `2003`：主营业务收入 `29.86 亿元`，净利润 `14.38 亿元`
- `2004`：主营业务收入 `61.74 亿元`，净利润 `30.39 亿元`
- `2005`：主营业务收入 `72.59 亿元`，净利润 `33.39 亿元`

这说明上市后的第一阶段，利润释放非常快，三峡机组投运、并表、调度能力提升，迅速把招股书逻辑变成现实业绩。

#### 3. 这阶段的投资意义

- 管理层说要扩容，真的扩了；
- 资产装入能转化为利润，而不是“只增规模不增收益”；
- 公司早期就建立了“招股书承诺可以兑现”的可信度。

### 阶段二：`2009-2014`
### 三峡资产进一步整合，现金牛属性强化

这一阶段的关键词是：**三峡资产进一步归位，上市平台的核心水电资产更加完整。**

公开资料显示：

- `2003/2005/2007`，公司分阶段收购三峡机组；
- `2009`，公司推进整体上市方案，收购剩余三峡机组及相关资产；
- 收购完成后，长江电力对三峡和葛洲坝核心资产的控制显著增强。

这阶段的本质变化是：

- 资产版图从“首批机组 + 葛洲坝”进一步扩大；
- 平台化色彩越来越强；
- 公司的盈利基础和分红基础更加稳固。

对价值投资者来说，这意味着公司开始从“成长型水电平台”逐步转向“成长 + 现金牛”双属性。

### 阶段三：`2015-2020`
### 溪洛渡、向家坝注入，第二次跃迁

这是长江电力上市后第二次关键跳跃。

#### 1. 重大资产重组

`2015` 年，公司公布重大资产重组预案，拟收购川云公司 `100%` 股权，核心资产为：

- 溪洛渡电站
- 向家坝电站

根据公开资料，重组于 `2016` 年完成。

#### 2. 重组效果

公开资料显示，完成后公司自有装机容量从约 `2527.7 万千瓦` 增至约 `4553.7 万千瓦`，增幅约 `73.6%`。

这件事很关键，因为它说明长江电力不是“上市后吃完三峡就没东西了”，而是仍然沿着招股书底层逻辑前进：**不断把三峡集团体系内的优质水电资产资本化、证券化。**

#### 3. 对分红逻辑的影响

这一阶段之后，长江电力的分红确定性明显上台阶。

公开资料显示，公司在 `2016` 年将未来十年的分红政策写入章程：

- `2016-2020`：每股现金分红不低于 `0.65 元`
- `2021-2025`：按不低于当年净利润的 `70%` 进行现金分红

这说明公司已经从“扩容中的成长股”，逐渐演变为：**以大水电资产为底层、以高分红为显性特征的公用事业核心资产。**

### 阶段四：`2021-2024`
### 乌东德、白鹤滩注入，六库联调成型

这是上市以来第三次决定性的跃迁。

#### 1. 乌东德、白鹤滩资产注入

公开资料显示：

- `2021` 年，三峡集团筹划将乌东德、白鹤滩资产注入长江电力；
- `2023` 年相关交易完成；
- 乌东德、白鹤滩自 `2023-01-01` 起纳入上市公司合并报表。

更准确地看，这轮注入经历了四个阶段：

`经营准备 -> 重组实施 -> 正式并表 -> 并表消化`

#### 2. 平台能力再上一个台阶

资产注入后，长江电力形成长江干流六座大型水电站统一平台：

- 乌东德
- 白鹤滩
- 溪洛渡
- 向家坝
- 三峡
- 葛洲坝

这意味着公司进入新阶段：

- 不再只是“单项目扩容”；
- 而是“六库联调、流域协同、系统级优化”。

#### 3. `2024` 的位置

根据公开业绩资料，`2024` 年长江电力实现：

- 营业收入 `844.92 亿元`
- 归母净利润 `324.96 亿元`

和上市早期相比，已经是完全不同量级的公司。这不是简单自然增长，而是“三峡平台化 + 溪向注入 + 乌白注入 + 长期运营优化”层层叠加后的结果。

## 三、把招股书、年报和分红串起来看

### 1. 招股书说的主线，后来都兑现了

招股书的主线是：

- 先买首批机组；
- 再持续收购后续优质机组；
- 不断扩大装机规模；
- 把装机扩大转化为收入、利润和现金流；
- 最终成为大型清洁能源平台。

从 `2003-2024` 看，这条主线并没有偏离。

### 2. 年报告诉我们的，是“兑现节奏”

年报最重要的作用，不是告诉我们“今年多赚了几亿”，而是帮助判断：

- 资产注入有没有兑现成盈利；
- 发电量增长有没有兑现；
- 电价和来水影响大不大；
- 公司有没有把平台做厚，而不是只做大。

### 3. 分红把商业模式最终坐实了

如果说招股书告诉你“这是一家会越来越大的公司”，那分红就是告诉你“这家公司长大以后，真的把钱分出来了”。

长江电力的长期投资价值，很大程度上正是因为它：

- 资产优质；
- 现金流强；
- 分红政策清晰；
- 实际分红执行度高。

## 四、长江电力的长期护城河，到底是什么

### 1. 稀缺流域资源

优质的大型水电站不是想建就能建。点位稀缺、审批复杂、资本开支巨大、建设周期超长，决定了后来者极难复制。

### 2. 资产平台地位

长江电力最强的地方，不只是“有一个好电站”，而是：**它是三峡体系优质水电资产持续证券化的核心平台。**

### 3. 低边际成本 + 长生命周期

水电站一旦建成：

- 边际成本低；
- 生命周期长；
- 现金流持续性强。

### 4. 六库联调带来的系统优势

到 `2023-2024`，长江电力的竞争力已不仅是“多几个机组”，而是：

- 流域级协同调度；
- 丰枯优化；
- 系统级运营效率。

这会让平台价值高于单站价值之和。

## 五、分红主线

### 1. 早期分红

公开资料显示，`2003 年度` 分红为每股 `0.09 元`，这是上市后最早的现金回报起点。

### 2. 制度化高分红阶段

根据公开资料和公司长期规划：

- `2016-2020`：每股现金分红不低于 `0.65 元`
- `2021-2025`：现金分红不低于当年净利润的 `70%`

这表明分红不再只是“有钱就分一点”，而是上升为资本配置承诺的一部分。

### 3. 高分红与平台扩容并行

长江电力最难得的一点在于：

- 一边继续大规模资产重组和资本开支；
- 一边还能维持高比例现金分红。

这说明它不是“增长和分红二选一”的公司，而是：**增长、现金流、分红三者可以共存。**

### 4. 到 `2024` 的分红观察

公开资料显示：

- `2022` 年现金分红比例一度达到高位；
- `2024` 年公司首次进行中期分红；
- 上市以来累计分红已超过 `1600 亿元`，部分公开统计口径已接近 `1880 亿元`。

最重要的不是精确到最后一位，而是趋势已经非常明确：**长江电力已经从“会增长的公司”，变成“能持续把增长兑现为股东现金回报的公司”。**

## 六、如果把 `2003-2024` 浓缩成一条投资主线

可以浓缩成下面这条链条：

`招股书平台逻辑 -> 三峡机组持续装入 -> 三峡资产进一步整合 -> 溪洛渡/向家坝注入 -> 乌东德/白鹤滩注入 -> 六库联调成型 -> 高分红现金牛`

这条链条说明，长江电力并不是简单“多年持有的一只股票”，而是一个：

- 资产持续升级
- 规模持续扩大
- 现金流持续增强
- 分红持续提高

的长期复利平台。

## 七、当前研究结论

### 1. 招股书最大的前瞻判断已经兑现

上市时如果真正看懂了招股书，会抓到一个核心判断：

**这不是一座电站的上市，而是一整条长江干流水电资产平台化的开始。**

后来的 `2003-2024` 证明，这个判断是对的。

### 2. 长江电力的价值，不只是“稳”

很多人把长江电力简单理解成高股息公用事业股，但长期看它其实有两层价值：

- 第一层：稳定现金流和高分红；
- 第二层：优质资产持续装入带来的平台扩容。

### 3. 真正的长期看点仍然没有变

到 `2024`，长江电力最重要的研究问题仍然是：

- 资产还有没有继续优化空间；
- 六库联调还能释放多少协同效率；
- 高分红政策能否长期维持；
- 在成熟期，它能否继续保持“现金牛 + 适度成长”的双重属性。

## 八、后续值得继续补的 5 个问题

1. `2009` 整体上市后，三峡资产完全归位对估值提升有多大？
2. 溪洛渡、向家坝注入完成后，ROE 与自由现金流发生了什么变化？
3. 乌东德、白鹤滩并表后，平台协同的利润释放节奏是否符合预期？
4. `2003-2024` 的历年分红表，是否可以再整理成一张完整的现金回报年表？
5. 未来十年，长江电力更像“高股息核心资产”，还是仍然具备平台型成长属性？

一句话总结：

**长江电力 `2003-2024` 的长期历史，本质上是一部“优质水电资产持续装入上市公司，并最终转化为高分红长期复利”的兑现史。**
