## 二、业界标准：价值投资机构如何分析财报

### 2.1 代表性研究机构及其方法论

| 机构 | 方法论特点 |
|---|---|
| **Morningstar** | 经济护城河评级 + DCF估值 + 不确定性评级，三位一体 |
| **Berkshire Hathaway / 巴菲特** | 关注长期ROE、自由现金流、管理层诚信、护城河持久性 |
| **格雷厄姆/多德学派** | 安全边际、净流动资产价值、盈利稳定性、低估值筛选 |
| **唐朝（手把手教你读财报）** | 三大报表逐项解读、附注深挖、多年对比、交叉验证 |
| **CFA Institute 框架** | 财务报表分析→比率分析→行业对比→估值建模→风险评估 |
| **Columbia Business School** | 2025年新书《Financial Statement Analysis for Value Investing》|

### 2.2 标准化8阶段研究流程

基于业界最佳实践，一个完整的价值投资财报分析流程包含8个阶段：

```
① 数据采集与聚合
    ↓
② 数据清洗与标准化
    ↓
③ 指标工程（比率计算）
    ↓
④ 分析与建模
    ↓
⑤ 风险评估
    ↓
⑥ 可视化与呈现
    ↓
⑦ 决策框架
    ↓
⑧ 回测与持续改进
```

#### 阶段一：数据采集与聚合
- **核心数据源**：年报PDF、交易所公告、标准化财务数据API、行业数据
- **另类数据**：管理层电话会议纪要、分析师研报、ESG数据
- **关键实践**：构建本地"数据湖"，缓存常用数据降低API依赖

#### 阶段二：数据清洗与标准化
- 通常占总分析时间的 **60-70%**
- 处理会计准则变更、科目重分类、单位换算、异常值检测
- 历史重述值修正（你的项目已经做了这一步！）

#### 阶段三：指标工程
- **盈利能力**：毛利率、营业利润率、净利率、ROE、ROA、ROIC
- **偿债能力**：资产负债率、利息保障倍数、债务/EBITDA
- **运营效率**：资产周转率、存货周转天数、应收账款周转天数
- **现金流质量**：经营现金流/净利润、自由现金流收益率、资本开支/折旧
- **估值**：PE、PB、EV/EBITDA、股息率、PEG
- **杜邦分析**：分解ROE为净利率×资产周转率×权益乘数

#### 阶段四：分析与建模
- **历史趋势分析**：5-10年纵向比较，识别拐点
- **同行对比**：横向比较行业内公司的关键指标
- **DCF估值**：自由现金流折现、股利折现
- **情景分析**：乐观/中性/悲观三种情景测算
- **质量评分**：Piotroski F-Score、Altman Z-Score等

#### 阶段五：风险评估
- 财务风险（杠杆、流动性）
- 业务风险（客户集中、供应链）
- 治理风险（关联交易、审计意见）
- 宏观风险（利率、汇率、政策）

#### 阶段六：可视化与呈现
- 交互式仪表盘
- 时间序列趋势图
- 财务健康"雷达图"
- 估值区间图

#### 阶段七：决策框架
- 预设买入/卖出阈值
- 清单驱动的确认流程
- 投资日志记录

#### 阶段八：回测与持续改进
- 记录每笔决策的逻辑和结果
- 定期归因分析
- 迭代优化研究模板

## 五、集成方案：分三期推进

### 第一期：补齐核心分析能力（P0）

#### 5.1 新增脚本：`scripts/compute_financial_ratios.py`

**目标**：基于已有的标准化CSV数据，自动计算全套财务比率

```
输入: 萝卜投研整理后的CSV（利润表、资产负债表、现金流量表、财务摘要）
     ↓
计算: 50+核心财务比率
     ↓
输出: 比率时序CSV + 比率Excel + 比率变化趋势图
```

**核心指标清单**：

| 类别 | 指标 | 公式来源 |
|---|---|---|
| 盈利能力 | 毛利率、营业利润率、净利率、ROE、ROA、ROIC | 利润表 + 资产负债表 |
| 杜邦分解 | 净利率 × 资产周转率 × 权益乘数 | 三表联动 |
| 偿债能力 | 资产负债率、利息保障倍数、流动比率、速动比率 | 资产负债表 + 利润表 |
| 现金流质量 | OCF/净利润、自由现金流、FCF收益率 | 现金流量表 |
| 成长性 | 营收增长率、净利增长率、经营现金流增长率 | 同比计算 |
| 分红 | 股息率、分红率 | 财务摘要 |
| 每股指标 | EPS、BPS、每股经营现金流、每股自由现金流 | 已有 |

**技术方案**：
- 复用 `萝卜投研-财务页下载/整理/csv/` 目录下已有的标准化CSV
- 用 pandas 计算所有比率，输出到 `整理/ratios/` 新目录
- 生成比率时序Excel（每个公司一个文件）和趋势图

#### 5.2 新增脚本：`scripts/build_valuation_model.py`

**目标**：提供DCF和相对估值的计算框架

```
输入: 比率CSV + 用户假设参数（增长率、折现率、永续增长率）
     ↓
计算: DCF估值 + PE/PB区间估值 + 敏感性分析
     ↓
输出: 估值报告Markdown + 敏感性矩阵Excel
```

**核心功能**：
- 简化版DCF：基于最近N年自由现金流，预测未来5-10年，折现求和
- 相对估值：基于历史PE/PB百分位，给出当前位置
- 敏感性分析：折现率 × 增长率的矩阵表

### 第二期：扩展数据源与可视化（P1）

#### 5.3 新增脚本：`scripts/download_akshare_finance.py`

**目标**：用AKShare作为萝卜投研的备份/补充数据源

```python
# AKShare 核心接口（免费、无需Token）
import akshare as ak

# 财务报表
df_income = ak.stock_financial_report_sina(stock="600900", symbol="利润表")
df_balance = ak.stock_financial_report_sina(stock="600900", symbol="资产负债表")
df_cashflow = ak.stock_financial_report_sina(stock="600900", symbol="现金流量表")

# 关键指标
df_indicators = ak.stock_financial_analysis_indicator(symbol="600900")

# 行业对比数据
df_industry = ak.stock_board_industry_cons_em(symbol="水电")
```

**优势**：
- 完全免费，无需Cookie
- 数据源丰富（新浪、东方财富、同花顺等聚合）
- 可获取行业内全部公司数据，支持同行对比

#### 5.4 升级可视化：引入 Plotly 交互式图表

**目标**：从matplotlib静态图升级为Plotly交互式HTML

```
现有: matplotlib → PNG图片
升级: Plotly → 交互式HTML（可缩放、悬停显示数值、导出）
```

可生成的交互式图表：
- 财务比率雷达图（多维度一览）
- 历史PE/PB估值区间带
- 杜邦分析瀑布图
- 自由现金流与分红对比时间线

#### 5.5 同行对比框架

**目标**：新建 `scripts/compare_peers.py`

```
输入: 多个公司的比率CSV
     ↓
分析: 横向对比关键指标排名
     ↓
输出: 同行对比报告 + 排名表 + 雷达图
```

例如，比较水电行业的：长江电力、华能水电、国投电力、川投能源

## 九、总结与建议

### 当前项目的定位

你的项目在个人价值投资研究系统中已经处于**中上水平**。与业界对比：

- **数据采集和清洗**：达到机构研究员的基础水平
- **PDF解析和对账**：超越大多数个人投资者的实践
- **深度研究笔记**：质量接近专业研报
- **分析计算和估值**：这是最需要补强的环节

### 下一步行动建议

1. **先做 `compute_financial_ratios.py`**：这是性价比最高的一步。有了比率时序数据，后续分析都有了基础。
2. **再做 `build_valuation_model.py`**：有了比率和估值，研究笔记就能从"定性分析"升级为"定量+定性结合"。
3. **安装AKShare作为备份数据源**：万一萝卜投研Cookie失效，不至于断数据。
4. **可视化升级可以和前面同步进行**：Plotly学习成本不高，效果提升明显。

**一句话结论**：当前项目的数据管道已经很强，接下来的重点是**在已有的高质量数据之上，补齐分析计算层和估值层**，让整个系统从"数据整理工具"升级为"完整的投资分析平台"。
