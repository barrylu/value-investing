# 基础工具搭建说明（2026-03-28）

## 这次已经搭好的工具

### 1. 财务比率计算
- 脚本：`scripts/compute_financial_ratios.py`
- 作用：从 `萝卜投研-财务页下载/整理/csv/` 自动生成财务比率时序、Excel 和年度趋势图
- 关键输出目录：`年报/[公司]/萝卜投研-财务页下载/整理/ratios/`

### 2. 估值模型
- 脚本：`scripts/build_valuation_model.py`
- 作用：基于年度比率结果生成 DCF 估值报告、敏感性矩阵，并支持可选 PE/PB 区间估值
- 关键输出目录：`年报/[公司]/萝卜投研-财务页下载/整理/valuation/`

### 3. AKShare 备份数据源
- 脚本：`scripts/download_akshare_finance.py`
- 作用：下载免费财务数据，补上一条不依赖 Datayes Cookie 的备份链路
- 关键输出目录：`年报/[公司]/AKShare-财务数据/`

### 4. 交互式图表
- 脚本：`scripts/plot_interactive_charts.py`
- 作用：把年度比率结果导出为 Plotly 交互式 HTML 仪表盘
- 关键输出目录：`年报/[公司]/萝卜投研-财务页下载/整理/charts/interactive/`

### 5. 横向对比框架
- 脚本：`scripts/compare_peers.py`
- 作用：读取多个公司的年度比率结果，生成对比表、排名表和雷达图
- 默认输出目录：`研究笔记/同行对比/`

### 6. 公共辅助模块
- 脚本：`scripts/value_investing_common.py`
- 作用：统一处理路径、期间排序、字段匹配、Markdown 表格输出等公共逻辑

### 7. 模板与说明
- 新模板：`模板与清单/估值分析模板.md`
- 已更新：`README.md`、`Python-虚拟环境使用指南.md`
